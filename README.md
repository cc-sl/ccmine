# 自动化扫雷


目的是开发一个自动化扫雷工具

目标是启动工具，自动把mineswifter完全解完，无需人工介入


## 预计开发流程

首先能够视觉识别扫雷局面，自动化录入（直接传入当前局势的截图或录像，预计使用opencv）

使用模板匹配视觉识别局势

然后控制鼠标坐标进行自动化左键点击已经标记好的雷区

之后自动识别一阶推理的未知雷区 右键标记

之后自动识别二阶推理的未知雷区 右键标记




## 预计适配页面

首先适配在线扫雷网站 [易解的扫雷](https://mineswifter.com/?lang=zh)

win10扫雷

win7扫雷

更古早的扫雷
